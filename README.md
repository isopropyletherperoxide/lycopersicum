# lycopersicum
Pomodoro Clock written in Rust

```
Welcome to lycopersicum! 0.3.1
Usage: lycopersicum [OPTIONS]

Options:
  -t, --timers <TIMERS>...  A list of timers (specified in minutes) to loop through
  -n, --no-notifications    Disable Notifications
  -h, --help                Print help

```
